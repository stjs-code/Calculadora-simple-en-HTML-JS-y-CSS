# Calculadora en HTML, CSS y JavaScript

Esta es una calculadora simple desarrollada como proyecto personal. Permite realizar operaciones básicas (+, -, ×, ÷) y tiene un diseño visual simple.

## Funcionalidades
- Suma, resta, multiplicación y división
- Interfaz amigable

## Cómo usar
Descarga o clona el repositorio y abre el archivo `index.html` en un navegador para probar la calculadora.

## ¿Por qué está hecha así?
Sé que hay mejores formas de hacer una calculadora, pero la hice según mi conocimiento. En este caso, me di cuenta de que no podía simplemente hacer que leyera distintas operaciones; sé que hubiera sido fácil usar un ParseInt y solucionarlo, pero quería realmente buscar una solución lógica. Utilicé la propiedad slice() para leer cada operación por separado, y aunque me costó un poco encontrarle una solución, lo logré con esfuerzo. Este es uno de mis primeros proyectos, y su única función es la de cumplir su trabajo y ser el inicio de mi progreso.
